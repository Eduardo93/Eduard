Para probar el proyecto debe estar instalado
•	MYSQL (el puerto puede ser diferente al de defecto, sólo para conectar al sistema deberá especificarse)
•	Java 
•	Eclipse
Crear una base de datos con el nombre “generadores” a partir del script del archivo “ScriptGeneradores”, en caso de tener problemas con el archivo, se adjunta el archivo “ModelGeneradores” para crear la base de datos desde workbench.
Para crear el proyecto en java, abrir eclipse y crear un nuevo proyecto.
Una vez creado el proyecto, seleccionar la opción para importar y seleccionar el directorio “generadores2014”, aquí es donde se encuentra el código fuente del sistema.
Una vez importado el proyecto, verificar la existencia de las bibliotecas utilizadas, si no existe la relación a las bibliotecas crear la importación de las bibliotecas en el directorio “generadores2014/lib/”.
